Activity
Every Screen is an activity
It is used to create any screen in android points to notice are
  • we extend >AppCompatActivity class to make it activity
  • we override onCreate(savedInstanceState: Bundle?) function of AppCompatActiviry class
  • finally we set the contest view of this particular activity using setContentView(R.layout.activity_main) here R.layout.activity_main is name of an xml file in layout directory of res (resource) folder
  • setContentView is an Activity method only. Each Activity is provided with a FrameLayout with id "@+id/content" (i.e. the content view). Whatever view you specify in setContentView will be the view for that Activity. setContentView() is used to set the layout of a particular activiry
class MainActivity : AppCompatActivity() { override fun onCreate(savedInstanceState: Bundle?) { super.onCreate(savedInstanceState) setContentView(R.layout.activity_main) } }

DatePickerDialog
It is used to show date picker in android here you explicitly need to create object of DatePickerDialog class
DatePickerDialog(this,{ _, yearNow,monthNow,dayNow -> do something},year,month,day)

  • it takes a total of 5 arguments
